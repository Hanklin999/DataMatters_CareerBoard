## Product Data Science analysis

The repository includes a reproducible Product DS analysis layer under [`analysis/`](analysis/README.md).

It contains:

- a deduplicated event staging view;
- a session-level funnel and outcome mart;
- question-friction diagnostics;
- recommendation-quality analysis;
- a privacy-safe local notebook workflow;
- explicit data-quality and product-decision rules.

Production impact metrics are not reported until the underlying event export passes the documented quality checks.
